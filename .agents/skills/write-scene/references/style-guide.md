# Стилістичний довідник

Цей документ визначає стилістичні правила для написання сцен у цьому проекті. Агент має дотримуватися цих правил при створенні та редагуванні тексту сцен.

## Базова структура прози

### Абзаци

- Кожен абзац — завершена думка, 3–7 речень
- Абзаци розділяються порожнім рядком
- Перший абзац сцени — без відступу
- Уникай «стін тексту»: абзаци довші 10 речень варто розбивати

### Діалоги

- Кожна репліка — з нового рядка
- Використовуй тире (`—`) для позначення прямої мови
- Слова автора після репліки — через кому, з маленької літери:

```markdown
— Я не знаю, про що ти говориш, — сказав Влад, відводячи погляд.
— Знаєш, — Марта зробила крок уперед. — І я це доведу.
```

- Якщо репліка переривається дією — використовуй два тире:

```markdown
— Я думав, що ти... — він затнувся, підбираючи слова, — ...що ти на моєму боці.
```

### Внутрішній монолог

- Прямі думки персонажа — курсивом:

```markdown
*Що ж я роблю?* — промайнуло в голові. *Треба було залишитися вдома.*
```

- Непрямий внутрішній монолог — звичайним текстом:

```markdown
Влад розумів, що запізнюється. Ще трохи — і все піде шкереберть.
```

## POV (точка зору)

### Персонажне POV (обмежена третя особа)

- Оповідь ведеться від третьої особи, але через сприйняття POV-персонажа
- Читач знає лише те, що знає, бачить і відчуває цей персонаж
- Не перестрибуй між головами персонажів у межах однієї сцени

```markdown
// Правильно: все через сприйняття Влада
Влад відчув холод. Марта здавалася напруженою — принаймні, так йому здалося.
```

```markdown
// Неправильно: стрибок у голову Марти
Влад відчув холод. Марта думала про те, як би швидше піти.
```

### Всезнаючий наратор

- Якщо сцена не має явного POV (поле `pov` порожнє) — можна використовувати всезнаючого наратора
- Але краще дотримуватися одного POV на сцену для глибшого занурення

## Тон і темп

### Темп

- **Екшн-сцени:** короткі речення, швидка зміна дій, мінімум описів
- **Діалогові сцени:** середній темп, чергування реплік і дій
- **Описові/атмосферні сцени:** довші речення, більше деталей, повільніший ритм

### Емоційне навантаження

- Показуй емоції через дії та фізичні реакції, а не називай їх прямо:

```markdown
// Краще (показ)
Кулаки Влада стиснулися. Він мовчав, дивлячись у підлогу.

// Гірше (розповідь)
Влад був дуже злий.
```

## Деталі та описи

### Правило трьох

При описі нового місця чи персонажа дай **2–3 яскраві деталі**, не більше. Читач домалює решту.

```markdown
// Достатньо
Старий млин стояв на пагорбі — чорний силует на тлі сірого неба. Вітер гойдав поламане крило, і воно рипіло, наче хтось кликав на допомогу.
```

### Сенсорні деталі

Задіюй різні органи чуття, не тільки зір:

- **Запах:** «Пахло вогкістю і прілим листям.»
- **Звук:** «Десь далеко завив собака.»
- **Дотик:** «Камінь був холодним і слизьким.»
- **Смак:** «У роті стояв металевий присмак крові.»

## Довжина сцени

- **Мінімальна сцена:** ~200 слів (короткий момент, перехід)
- **Середня сцена:** 500–1000 слів (стандартний епізод)
- **Довга сцена:** 1500–2500 слів (ключова подія, кульмінація)

## Wikilinks у тексті сцени

- Wikilinks мають бути **непомітними** при читанні
- Перше входження персонажа/локації/події в сцені — `[[wikilink]]`
- Друге і наступні входження в тій самій сцені — **без wikilinks**, звичайний текст
- Приклад:

```markdown
[[vlad]] зайшов до [[old-mill]]. Всередині було темно. Влад (без wikilinks)
пройшов у глибину приміщення...
```

## Форматування діалогів: розширені правила

### Репліка + дія

Якщо репліка супроводжується дією того ж персонажа — дія в тому ж рядку:

```markdown
— Я не вірю тобі. — Влад схрестив руки на грудях.
```

### Репліка + дія іншого персонажа

Якщо дія належить іншому персонажу — новий рядок:

```markdown
— Я не вірю тобі.
Марта зітхнула.
— Твоє право, — сказала вона. — Але це правда.
```

### Перерваний діалог

Три крапки для незавершеної репліки:

```markdown
— Я хотів сказати, що...
— Що? Що ти хотів сказати?
```

## Що уникати

- **Інфодампінгу:** не викладай усю передісторію в одній сцені через діалог чи опис
- **Кліше:** «він глибоко зітхнув», «серце шалено калатало», «кров застигла в жилах»
- **Надмірних прислівників:** «сказав він сердито» → краще показати сердитість через дію
- **Пасивного стану:** «двері були відчинені» → «двері відчинилися» або «хтось відчинив двері»
- **Пояснень:** не пояснюй читачу те, що він може зрозуміти з контексту
